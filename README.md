# UniQ Te Herenga Waka - Brand Identity System

The official brand kit for **UniQ Te Herenga Waka**, the LGBTQIA+ student organisation at Victoria University of Wellington, Aotearoa New Zealand.

## Live Site

[brand.uniqthw.org.nz](https://uniqthw.github.io/brand-kit/)

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Main brand kit document |
| `styles.css` | All styles and design tokens |
| `main.js` | Nav highlighting and scroll animations |

## Usage

The brand kit is a single-page static site. Clone the repo and open `index.html` in any browser, or serve it with any static host.

```bash
git clone https://github.com/[org]/brand-kit.git
cd brand-kit
# Open index.html in your browser, or use a local server:
npx serve .
```

## Contributing

This is a private repository for the UniQ Te Herenga Waka committee and design team.

1. Clone the repository (do not fork — this is closed source)
2. Create a branch (`git checkout -b fix/your-fix`)
3. Commit your changes following the existing code style
4. Open a pull request — all changes require review from the Social Media Officer or Publications Officer before merging
5. Never share repository access with anyone outside the committee without approval
